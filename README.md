# ChatOnline

#### 介绍
多用户在线聊天系统，gin+mysql+react

#### 计划接口
- login（登陆）
- register（注册）
- forget（重置密码）
- email（发送和验证邮箱验证码）
- websocket（登陆过后的所有操作通过此完成）


#### 后端使用的框架或者包（初步成型）
- Gin（web框架）
- Goini（配置文件）
- Gorm（数据库）
- Go-jwt（token验证）
- Logrus（日志管理）
- Lfshook（配合logrus进行日志管理）
- File-rotatelogs（日志分割）
- Validator（数据验证）
- Scrypt（密码存储采用的加密）
- Cors（跨域）
- Websocket(核心，通信使用)
